# SQLCipher compatibility experiment

Этот пакет проверяет точную unreleased-ревизию официального репозитория
`rusqlite`, в которой bundled SQLCipher обновлён до 4.17.0.

Проверяется:

- фактическая версия SQLCipher;
- raw key без внутреннего PBKDF2;
- отсутствие контрольного plaintext в файле;
- повторное открытие правильным ключом;
- одинаковый закрытый отказ чтения и изменения схемы с неверным ключом;
- обычная plaintext SQLite-база для открытой транспортной очереди.

Эксперимент не доказывает безопасность SQLCipher или Lantern. Он проверяет
только совместимость выбранной версии с требуемым способом сборки и API.

Зафиксированный прогон и его ограничения находятся в
[`RESULTS.md`](RESULTS.md).
